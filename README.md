
Plataforma digital inclusiva para reducir la brecha digital educativa en comunidades vulnerables de Bogotá — Prototipo (Morado + Negro)

Este repositorio contiene un **prototipo funcional (TRL5)** del proyecto: Plataforma digital inclusiva para reducir la brecha digital educativa en comunidades vulnerables de Bogotá.

Presentado por: Daniel Arturo Velásquez, Katherine Alfonso Villalba, Camilo Andrés Romero — Universidad Nacional Abierta y a Distancia - UNAD

Contenido
- index.html — Página principal / landing del prototipo.
- estudiante.html — Panel de estudiante (simulado).
- docente.html — Panel de docente (simulado).
- assets/styles.css — Estilos (tema morado/negro).
- assets/script.js — Interacciones simuladas.
- Fase_Ingenieril_Prototipo.txt — Texto listo para pegar en el documento maestro (UNAD).

Cómo usar
1. Descargar o clonar el repositorio.
2. Abrir index.html con cualquier navegador.
3. Navegar entre Inicio / Estudiante / Docente.

Instrucciones para subir a GitHub están al final del documento.
